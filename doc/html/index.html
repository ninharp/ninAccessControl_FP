MagicLight Controller Main Unit

MagicLight is a Controller for WS2811 + WS2812B based LED Strips

The MagicLight can be controlled over an Android App over Bluetooth or over a dedicated Remote Control (chooseable between cable bond and bluetooth version)

MagicLight includes several LED effects which let you change a lot of settings to get individual show

Additionaly you can connect your radio oder plug in an MP3 player to get the special music activated effects running

You can change various settings, like saturaton, hue, brightness, ...

Hardware:


Nucleo-F103RB Board with

STM32F103RB ARM 32 Bit CPU

on Own Faceboar in sandwich method

Pin Usage:


Port A 6 / TIM3_CH1 - LED Data Channel 1
Port A 7 / TIM3_CH2 - LED Data Channel 2 !!!
Port B 0 / TIM3_CH3 - LED Data Channel 3
Port B 1 / TIM3_CH4 - LED Data Channel 4


Port B 5 - LED Interrupt Channel 1
Port B 6 - LED Interrupt Channel 2
Port B 7 - LED Interrupt Channel 3
Port B 8 - LED Interrupt Channel 4


Port A 9 / USART1_TX - Bluetooth RX
Port A 10 / USART1_RX - Bluetooth TX
Port A 8 - Bluetooth Key Line


Port B 10 / I2C2_SCL - Remote I2C SCL
Port B 11 / I2C2_SDA - Remote I2C SDA


Port A 2 / ADC12_IN2 - Show / Abblendlicht
Port A 3 / ADC12_IN3 - Filter H
Port A 4 / ADC12_IN4 - Filter M
Port A 5 / ADC12_IN5 - Filter L
Port A 11 - Tachometer



DEBUG:
Port C 13 - Debug Key B1 on Nucleo Board
Port C 7 - Debug Green LED (temporarily)
