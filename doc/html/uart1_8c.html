Detailed Description

This file includes the UART1 functions It handles ML Protocol CRC Check and general parsing of the MagicLight Protocol.

Author

Date
14.09.2014 - erste Version
Note
Packet Format: STX | LEN | ADR | DATA | BCC | 0x0D (\r) | 0x0A (\n)

Adress Byte wird wie folgt zusammengesetzt:
 Bit 7  6  5  4  3  2  1  0
                 |–|–|–|— Typ des Telegramms (1=General / 2=Effektliste / 3= Einstellungen / ...) (one nibble)
              |.-----------— Fehler (Bit 4)
           |---------------— End condition (Bit 5)
        |------------------— Start Condition (Bit 6)
     |---------------------— Antwort auf Anfrage oder Broadcast (1= Antwort / 0= Broadcast) (Bit 7)
